# ESGenie 본선 시연영상 — 컷 단위 시나리오

> 형식: 발표 없음, **영상만 제출** → 화면 자막(on-screen text)이 내레이션을 대신함.
> 길이 목표: 3~3.5분. 두 장면(상장사 공시 자동화 / 협력사 실사 검증)으로 제품 두 축을 보여준다.
> 촬영은 `ESGENIE_STRICT=1 streamlit run app.py` 라이브 화면.

---

## 핵심 메시지 (영상이 증명할 것)
1. **ESG 보고서 없는 상장사** — DART 정기공시를 자동 흡입해 K-ESG 공시를 생성하고, **공시 공백을 즉시 진단**.
2. **DART조차 없는 비상장 협력사** — 현장 증빙으로 직접 검증하고, **협력사 자가주장 vs 증빙 불일치(그린워싱)를 자동 적발 🚩**.

## 법적 안전 규칙 (촬영 내내 준수)
- 장면1 회사는 **"코스피 상장 A사"로 익명** (실명·로고·사업자번호 화면 노출 금지). DART 수치는 공개데이터라 그대로 OK.
- 장면1엔 **부정 결론 금지** — "공시 격차 진단"(사실)만. 🚩(허위 적발)는 **가상 한울정밀에만**.
- 한울 증빙엔 "시연용 가상자료" 워터마크 유지(이미 적용됨).

---

## 사전 준비 체크 (녹화 전 1회)
- [ ] `.env`: `DART_API_KEY`, `AZURE_DOC_INTEL_*`, `AZURE_OPENAI_*` 채워짐
- [ ] `python -c "import sentence_transformers"` (임베딩 sbert 정상 — 해시폴백이면 D3 품질↓)
- [ ] 화신 익명화 동작: 검색 시 "코스피 상장 A사"로 표시되는지 (`demo_aliases.py`)
- [ ] 한울 증빙 세트 경로 확인: `시연증빙세트_한울정밀공업/` (01~05)
- [ ] `ESGENIE_STRICT=1`로 실행 (mock 폴백 시 예외 → 라이브 보장)

## 녹화 순서 팁
- 장면1·2는 **각각 따로 실행**(사이드바 설정이 다름)해서 클린하게 찍고 편집에서 잇기.
- 클라이맥스(2.4)는 마우스 호버/확대로 🚩 배지와 "92% ↔ 29.3%" 텍스트가 또렷이 보이게.
- bbox 하이라이트(2.3)는 증빙 미리보기에서 박스가 뜨는 순간을 슬로우로.
- 자막은 한국어 + (선택) 영어 병기하면 심사 가독성↑.

## 미정/확인 필요
- 장면1 영역을 G만 vs E·S·G 전체로 보여줄지 (G만 = 자동채움 3개가 또렷, 전체 = 커버리지 격차가 더 드라마틱).
- 2.5 다중양식(현대차/KAP) 토글이 UI에 실제 노출되는지 — 안 되면 SAQ5 엑셀 하나로.
