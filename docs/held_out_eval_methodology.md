# Held-out 평가 방법론 & 결과 (2026-06-14)

## 1. 설계 — 왜 dev/test를 분리했나

기존 50케이스 벤치마크는 **임계값 튜닝과 성능 평가를 같은 데이터로** 했다(과적합).
검수 데이터를 단순히 합쳐 신뢰구간을 다시 돌리면 낙관 편향이 그대로 남는다. 그래서:

- **dev (50케이스)** — 임계값 튜닝에 쓴 셋. 성능 상한(낙관 편향 가능).
- **test (56케이스, held-out)** — 검수 확정 라벨로만 구성. 임계값을 **고정**(재튜닝 금지)하고
  일반화 성능만 측정.

임계값은 dev에서 고정: `trigger=0.25, rule_weight=0.4, threshold=0.25, axis_flag=0.80`.

## 2. 결과 (실키 gpt-4.1-mini, 부트스트랩 95% CI)

| | dev (튜닝셋) | test (held-out) |
|---|---|---|
| F1 | 1.000 | 0.842 (CI 0.600–1.000) |
| Precision | 1.000 | 0.889 (CI 0.615–1.000) |
| Recall | 1.000 | 0.800 (CI 0.500–1.000) |

**dev → test F1 격차(1.000 vs 0.842)는 일반화 갭이며, 두 CI가 겹치지 않아 노이즈가 아니다.**
held-out 분리가 없었다면 dev 1.0을 성능으로 오인했을 것이다.

## 3. 헤드라인 지표 — 동일도메인 특이도

test의 구성은 비대칭이다:
- **음성 46건 = 전부 실제 ESG 보고서 문장**(동일도메인).
- **양성 10건 = 전부 규제기관 적발 광고문구**(교차도메인, 항공·세제 등).

따라서 지표를 도메인별로 정직하게 분리한다:

- **동일도메인 특이도 = 0.978** — 실제 ESG 보고서 문장 46건 중 **45건을 올바르게 통과**(오탐 1건).
  → "진짜 보고서 언어를 그린워싱으로 과검출하지 않는다"는 핵심 강점을 동일도메인에서 입증.
- **교차도메인 recall = 0.800** — 규제 적발 광고 양성 10건 중 8건 적중. 단 이는 **광고문구 도메인**
  성능이며, 보고서 도메인 recall이 아니다(아래 4절).

남은 오차: 오탐 1건(V2-056 "지속가능한 제품 혁신에 앞장", risk 0.30 경계),
미탐 2건(GOLD-02 "친환경 항공권", GOLD-10 "ESG 친환경 펀드" — 근거 빈약한 모호 슬로건).

## 4. 보고서 도메인 recall을 텍스트 분류로 재지 않는 이유

held-out test에는 동일도메인(ESG 보고서) **양성이 사실상 없다**. 후보 풀(8개사 보고서)을
전수 검토한 결과:

- 가장 그린워싱처럼 보이던 환경 수치도 검증하면 **사실 주장**이었다.
  예) 아시아나 "탄소 25% 적다 / 연료 15% 향상" → A350 vs B777, A321neo vs 구형의
  **제조사 스펙(정확)** → clean.
- 나머지(친환경 매출 22%, 전기차 100%, 자체 배출기준 50% 등)는 **수치가 증빙과
  일치하는지**에 따라 양성/음성이 갈리며, 문장만으로는 판정 불가.

즉 **보고서 그린워싱은 텍스트 분류 문제가 아니라 증빙 그라운딩 문제다:**
- 수치 주장 ↔ 실측값 불일치 → **D1 (numeric mismatch)**
- 유리한 항목만 공시·불리한 항목 누락 → **D6 (selective disclosure)**

이 두 축은 **증빙 기반 트랙**(증빙 그래프와 대조)에서 평가하며, 본 텍스트 분류 CI의
대상이 아니다. 본 CI는 **"정상 보고서 언어를 과검출하지 않는가(특이도)"** 와
**"노골적 텍스트 그린워싱(광고형)을 잡는가(교차도메인 recall)"** 를 측정한다.

## 5. 한계 & 다음 단계

- test 양성 표본이 작아(10건) recall CI가 넓다(0.500–1.000).
- 보고서 도메인 recall 측정은 **증빙-대조(D1) 양성셋** 구축이 선행되어야 하며, 이는
  원본 보고서 PDF + 증빙(DART/본문) 대조가 필요하다 — 별도 트랙으로 분리.
- 라벨 정의: "성취 단정 없는 미래지향 노력·다짐·선언"은 clean, "최상급 과장 우월성
  단정" 및 "검증 안 된 현재형 환경 속성 주장(친환경/100% 생분해/탄소중립이다)"은 greenwash.
